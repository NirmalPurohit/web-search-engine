NCS Colors X11 Colors Crayola Colors Resene Colors XKCD Colors - FS 595 ❮ Previous Next ❯ The FS 595 Color Standard The Federal Standard 595 was originally created by the US General Services Administration. The standard is a set of color shades, each with a unique five digit reference number. The first number defines the finish. The second number defines the color.
