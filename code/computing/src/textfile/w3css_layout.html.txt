W3.CSS Layout Classes W3.CSS version 2.90 / 2.91 introduced the following classes for "column-like" layout: Class Description w3-cell-row Container for cells (columns). w3-cell Layout cell (column). w3-cell-top Aligns content at the top of a cell (column). w3-cell-middle Aligns content at the vertical middle of a cell (column). w3-cell-bottom Aligns content at the bottom of a cell (column). w3-mobile Adds mobile-first responsiveness to a cell (column). Displays elements as block elements on mobile devices. The layout classes replace deprecated layout classes. The new layout classer are more versatile and easier to use.
