Modal JS Popover JS Scrollspy JS Tab JS Tooltip Bootstrap Text/Typography ❮ Previous Next ❯ Bootstrap's Default Settings Bootstrap's global default font-size is 14px, with a line-height of 1.428. This is applied to the <body> element and all paragraphs (<p>). In addition, all <p> elements have a bottom margin that equals half their computed line-height (10px by default).
